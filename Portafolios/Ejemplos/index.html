Ejemplo 1: Formularios

Completa los datos

Validación de formularios

Por favor, rellena el formulario que se muestra a continuación. No contiene base de datos, por lo que tus datos no se guardarán en ningún sitio, pero permitirá mostrar una validación simple. De ser posible, ingresa algunos datos erróneos, presiona el botón "Registrarse" con los campos vacíos o intenta ingresar una fecha dónde indiques que eres menor de edad.

Fue creado enteramente con HTML, CSS y JAVASCRIPT nativos, haciendo uso de propiedades de ECMASCRIPT, CSS-GRID y variables CSS.

También se implementó el uso de expresiones regulares sencillas, que permiten realizar una validación de todos los campos correctamente, verificando que los datos ingresados por el usuario sean correctos.

const Expresiones = {

}

El uso de condicionales, funciones switch y métodos forEach son importantes en la validación de este formulario a la hora de que el código sea optimizado.

Ejemplo 2: Carrusel

Carruseles automáticos y estáticos.

Estos carruseles se ejecutan de tres formas diferentes. El primero está programado para realizar transiciones automáticas, mientras que el segundo y tercero esperan a que el usuario realice una interacción con ellos. Los tres son infinitos, por lo que cada presentación podrá mostrarse todo el tiempo que el sitio web lo requiera.

Los dos primeros tienen una programación muy sencilla, la cuál no le exige mucho rendimiento al sitio web. Esta programación hace que cada carrusel entre en bucle mediante variables cuyo valor es un número y al presionarse un botón, su valor aumenta o decrementa según se necesite o hasta que el tiempo especificado así lo determine.

El tercer carrusel utiliza una librería de "Materialize", omitiendo el resto de estilos que le otorga sus propiedades a todo el sitio web. Solo se tomó el fragmento de código que controla el carrusel. Estas propiedades son ideales para páginas publicitarias, de comida u hotelería según se convenga, pues su diseño práctico y de bajo consumo de resursos es ideal para una buena experiencia del usuario.

Ejemplo 3: Partículas

Partículas JS

Partículas JS es una librería de dominio público. Esta permite darle vida a tus sitios web con efectos controlables de bajo consumo para el rendimiento de tu sitio web.

Este control se te otorga mediante el archivo "Particles-JS" con cada una de las propiedades que puedes o quieras utilizar.

Su uso es muy similar al de materialize, con la diferencia de que esta librería se centra únicamente en la creación y control de partículas dentro de un sitio web. Pueden agregarse en cualquier parte que se desee personalizar

Ejemplo 4: Vídeo JS

Video JS

Vídeo JS es otra librería muy práctica y sencilla de usar. Permite personalizar nuestro propio reproductor a nuestro antojo, manipulando botones, barras, tiempo de carga, el control de volumen y un montón de propiedades más.

Consta de dos archivos. El control en Javascript y un archivo CSS para personalizar los estilos por defecto. El resto consiste en agregar un archivo de vídeo común a la etiqueta VIDEO en el html, para posteriormente agregar las clases correspondientes a la librería.

Es excelente a la hora de mostrar contenido audiovisual dentro de un sitio web, sin tener que depender de reproductores externos o que consuman mayor rendimiento.
